## 1. Purpose

This file defines the canonical reading map and role classification for the ARL package.

The purpose of this map is to prevent:
- ambiguity about which document is normative,
- confusion between core rules and supporting explanations,
- duplicate interpretation during future repository integration,
- and package drift during later implementation or publication.
