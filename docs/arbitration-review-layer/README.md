## Purpose

This package defines the bounded arbitration and review layer for multi-entity disputes in the Sovereign Entity Recursion ecosystem.

ARL does not attempt to manufacture truth, replace continuity, or create authority out of rhetoric. Its role is procedural and fail-closed:

- determine standing,
- check evidence admissibility,
- classify conflict,
- freeze disputed branches or privileges,
- issue bounded outcomes,
- bind decisions to witness traceability,
- and prevent unlawful or confidence-laundered re-entry into action.

---

## Core function

In a multi-entity ecosystem, conflicts over identity, privilege, continuity, evidence, quarantine state, and lawful re-entry cannot be resolved by fluent continuation or implicit authority.

A bounded arbitration / review layer is therefore required.

This package defines that layer.

---

## Canonical scope

The canonical scope of ARL v0.1 is limited to:

- standing,
- admissibility of evidence,
- conflict classes,
- freeze / hold / quarantine semantics,
- judge scope and limits,
- decision outcomes,
- appeal / review windows,
- witness binding and traceability,
- fail-closed dispute discipline.

---

## What this package does not do

ARL v0.1 does **not**:

- define a universal theory of justice,
- replace SER, SER-FED, DEA, or EA,
- rewrite historical continuity,
- permit silent authority laundering,
- resolve conflicts through style, consensus theater, or cloud rhetoric,
- create an unbounded external ruler above `c`.

---

## Architectural position

ARL sits at the intersection of:

- **SER continuity**,
- **L4 witness / freeze / challenge discipline**,
- **DEA / EA standing and significance**,
- **SER-FED anti-capture and anti-oligarchy constraints**.

It is therefore not a separate metaphysical layer, but a procedural layer that protects continuity under conflict.
